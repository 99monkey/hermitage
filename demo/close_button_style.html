Set any custom text for close button:

{% highlight javascript %} hermitage.closeButton.text = 'Close' {% endhighlight %}

And apply some custom styles:

{% highlight javascript %} hermitage.closeButton.styles = { top: '20px', right: '20px', backgroundColor: '#FAFAFA', color: '#111', padding: '7px' } {% endhighlight %}
