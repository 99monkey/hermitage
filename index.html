Overview

Hermitage is a Ruby gem for generation of image galleries (thumbnails and original images viewer).

Requirements

Hermitage requires Ruby on Rails version >= 3.1 with support of jQuery and CoffeeScript (jquery-rails and coffee-rails gems, respectively).

Quick Start

Gemfile {% highlight ruby %} gem 'hermitage' {% endhighlight %} Terminal {% highlight bash %} $ bundle $ rails generate hermitage:install {% endhighlight %} View {% highlight ruby %} render_gallery_for @images # or any other array of objects with image attachments {% endhighlight %}

It is enough in theory.
