Ruby gem for generation of image galleries

Dependencies

Rails

>= 3.2

Coffee-Script

gem coffee-rails

jQuery

gem jquery-rails

Quick Start

Gemfile 
gem 'hermitage'
Terminal 
$ bundle
$ rails generate hermitage:install
View 
render_gallery_for @images # or any other array of objects with image attachments

It is enough in theory. Try demo or read documentation for more info.
